Bubble Sort


Large values sorted first

A very important concept of algorithm

Best time Complexity is O(n)

Average and Worst time complexity is O(n²)

Insertion Sort


Efficient for smaller datasets

Effective for partially sorted array

Best time Complexity is O(n)

Average and Worst time complexity is O(n²)

Merge Sort


Dividing an array into smaller subarrays, sorting each subarray,

and then merging the sorted subarrays

Best time Complexity is O(nlogn)

Average and Worst time complexity is O(nlogn)

Seletion Sort


Takes the smallest element in an unsorted array and bringing it to the front

and bringing it to the front

Best time Complexity is O(n²)

Average and Worst time complexity is O(n²)
